
# Full Stack Chat Application with Voice Notes Search Integration using Next.js, Socket.io, Node.js

This is the complete Source Code for the chatlosss built using Next.js, Socket.io, Node.js, Firebase, Tailwind CSS, Prisma, PostgresSQL, and many more awesome technologies with Voice Call and Video Call Functionality.




## Installation

Setup Client

```bash
  cd client
  yarn
```

Setup Server

```bash
  cd server
  yarn
  npx prisma init
  npx prisma generate
```
